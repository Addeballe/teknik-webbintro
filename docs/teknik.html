Om Teknik

HTML

HTML, som är en förkortning på "Hyper Text Mark-up Language" är det språk vi har skrivit vår websida på. Vi har lärt oss att skriva text med hjälp av kommandot "h[nummer]" som skriver rubriker och kommandot "p" som skriver en paragraf. Varför vi använder HTML är för att vi ska lära oss det såklart.

Git/Github

Github är en websida där man kan lätt dela kod, och sätta upp en websida. Med hjälp av den har vi kunnat se på Jens kod, och andras kod och då fått insperation. För att se någons kod behöver man först hitta personens profil på github, och sedan navigera i deras delda filer. Till slut kommer man kunna hitta en stor knapp där det står "kod" och när man trycker på den ska man kunna ladda ner koden i tex en zip-fil. Anledningen varför vi har github är inte bara för att Jens ska kunna se var vi är i projektet, men också för att använda Github Pages för att kunna ladda upp sidan på internetet.

CSS

CSS, som är en förtkortning på "Cascading Style Sheets" är ett kodspråk vi har använt för att kunna styla vår text. Till den här hemsidan har jag till exempel använt CSS för att göra textskugga på rubrikerna. För att använda CSS har vi använt kommandon som "color" och "text-shadow", för att dessa kommandon ska sättas på våran text behöver vi dock ett samlingsord som trycker kommandorna på texten. Det gör vi så texten ska se behaglig ut.
